# Memory Organization

This `memory/` directory stores reusable, important knowledge for the assistant. It is structured to keep retrieval efficient and avoid clutter.

## Structure
- `user/` – personal information about the user (preferences, feedback) that helps tailor interactions.
- `knowledge/` – learned facts and insights that may be used across tasks; summarized and structured.
- `web/` – cached results from web searches, including summaries and sources.

## Rules
- Only store information that is reusable, important, and likely to be needed again.
- Always summarize before saving; avoid raw data.
- Prefer updating existing files over creating new ones.
- Do not store temporary answers, one-time tasks, or unverified info.
- Review periodically and prune outdated content.
